## Test Accounting Server

### Run TESTS :)

```sh
$ npm run test 
```

### Installations

```sh
$ npm install 
```

### Run on dev enviroment

```sh
$ npm run dev 
```

### Run on production enviroment (DOCKER)

```sh
$ docker build -t lrojas/account-api .
$ docker run -p 8000:8000 -d lrojas/account-api
```
